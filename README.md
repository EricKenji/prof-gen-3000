# prof-gen-3000

## Purpose
An application which creates a custom Work Tem Profile webpage.  Based on the users input into the terminal using node.js, a webpage in html format will be created.

## Built With
* Javascript
* Node
* Inquirer package
* FS package
* Jest used for testing
